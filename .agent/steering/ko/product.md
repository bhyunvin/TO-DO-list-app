# 제품 개요

현대적인 웹 기술로 구축된 풀스택 TO-DO List 애플리케이션입니다. 이 애플리케이션은 사용자 인증, 날짜 기반 todo 관리, 파일 업로드 기능 및 AI 지원 기능을 제공합니다.

## 주요 기능

- JWT 기반 stateless 인증을 통한 사용자 등록 및 인증
- 사용자 등록 시 개인정보 동의 필수
- 날짜 기반 todo 구성 및 관리
- Cloudinary 클라우드 스토리지를 사용한 파일 업로드 기능
  - 진행 상황 추적
  - 서버 사이드 파일 검증 (크기, 형식, 보안)
  - 프로필 이미지 및 todo 첨부파일 지원
- 함수 호출 기능을 갖춘 Google Gemini를 사용한 AI 챗 어시스턴트 (todo 읽기, 생성, 수정 가능)
- 프로필 이미지 지원을 포함한 사용자 프로필 관리
- 비밀번호 변경 기능
- Contact Developer 기능 (관리자에게 문의 이메일 전송)
- 사용자 선호도가 유지되는 다크 모드 테마 토글
- IP 익명화 스케줄러를 포함한 모든 작업에 대한 포괄적인 감사 로깅
- Bun.password (bcrypt 알고리즘)를 사용한 안전한 비밀번호 암호화 및 데이터 암호화 (AES-256-GCM)
- 환경 변수 기반 자격 증명 관리
- XSS 보호 기능을 갖춘 마크다운 렌더링

## 대상 사용자

AI 지원 및 파일 첨부와 같은 고급 기능을 갖춘 개인 작업 관리 시스템이 필요한 개인 사용자

## 아키텍처

이 애플리케이션은 JWT 기반 stateless 인증을 통해 REST API로 통신하는 별도의 프론트엔드 및 백엔드 애플리케이션을 가진 모노레포 구조를 따릅니다.
